## Full Description

**DevPlaybook** is a collection of 40+ free online developer tools built around one principle: get out of your way.

Every tool runs entirely in your browser. No data is ever sent to a server. No account required. No cookie banners. No ads. Just fast, private tools that work.

### What's included (42 tools across 8 categories)

**Data & Formats**
- JSON Formatter & Validator
- JSON to CSV, JSON to YAML, YAML to JSON
- JSON Schema Generator
- UUID Generator, Timestamp Converter, Number Base Converter

**Code & Testing**
- API Tester (Postman-lite, browser-only)
- Regex Tester with real-time match highlighting
- Diff Checker, JS Minifier, CSS Minifier, HTML Formatter
- Code Screenshot Generator

**Security & Encoding**
- JWT Decoder & Inspector
- Hash Generator (MD5, SHA-256, SHA-512)
- Base64 & URL Encoder/Decoder
- Password Generator, Chmod Calculator

**Content & Markdown**
- Markdown Preview & Editor
- Markdown to HTML, HTML to Markdown
- Lorem Ipsum Generator, Text Case Converter
- GitHub README Generator, Meta Tag Generator, OG Image Preview

**Design**
- CSS Gradient Builder
- Color Picker & Converter (HEX/RGB/HSL)
- Box Shadow Generator, Border Radius Previewer
- Flexbox Playground

**DevOps**
- Cron Expression Builder (visual, with next-5-runs preview)
- Docker Run → Compose Converter
- .gitignore Generator

**AI Tools**
- LLM Token Counter (GPT-4, Claude, Llama)
- Prompt Length Checker

**Pro AI Tools ($9.99/mo)**
- AI Code Review
- AI Doc Generator (JSDoc / Python docstrings)
- AI SQL Builder

### Why developers use DevPlaybook

- **Zero friction:** no signup, no install, no cookie consent
- **Private by design:** 100% client-side JavaScript — your data never leaves your machine
- **Fast:** Astro + Preact architecture, sub-second page loads, Lighthouse 90+
- **Bookmarkable:** every tool has its own clean URL
- **Mobile-friendly:** works on any device

### Tech stack

Astro + Preact + Tailwind CSS, deployed on Cloudflare Pages. Average Lighthouse score: 90+.
