你有沒有過這種經驗：每天花大量時間回覆客戶訊息、整理表單資料、寫社群貼文？這些重複性工作其實可以交給 AI 自動處理。這篇教你用 Claude AI 加上 n8n 自動化平台，打造專屬的 AI 工作流。

## 什麼是 n8n？

n8n 是一個開源的工作流自動化平台，你可以把它想像成一個「數位流水線」——把不同工具串接起來，讓資料自動流動處理。

**跟 Zapier 的差別**：n8n 可以自架（免費）、節點數量不限、支援更複雜的邏輯分支。

## 什麼是 Claude AI？

Claude 是 Anthropic 開發的 AI 模型，擅長理解長文本、生成高品質內容、遵循複雜指令。在自動化流程中，Claude 負責「思考」的部分——分析資料、生成回覆、做出判斷。

## 三個實用的入門工作流

### 工作流 1：自動回覆客戶常見問題

**流程**：客戶發送訊息 → n8n 接收 → Claude 判斷問題類型 → 自動生成回覆 → 發送給客戶

**適用場景**：LINE 官方帳號、Email 客服、網站聊天機器人

**設定重點**：
- 在 Claude 的 System Prompt 中放入你的業務資訊（營業時間、價目表、常見 QA）
- 設定關鍵字過濾，只有符合條件的訊息才會觸發 AI 回覆
- 加入人工審核節點，重要問題轉給真人處理

### 工作流 2：自動生成社群貼文

**流程**：排程觸發 → 從資料庫取素材 → Claude 生成貼文 → 發布到社群平台

**適用場景**：Instagram 文案、Facebook 貼文、部落格文章

**設定重點**：
- 提供品牌語調範例給 Claude（例如：「活潑但專業，使用繁體中文」）
- 每次生成 3～5 個版本，人工挑選最好的
- 搭配排程節點，設定每週自動生成固定數量的內容

### 工作流 3：表單資料自動整理

**流程**：Google Form 提交 → n8n 接收 → Claude 分類和摘要 → 寫入 Google Sheets → 通知負責人

**適用場景**：預約表單分類、問卷回覆分析、報名資料整理

**設定重點**：
- 讓 Claude 按照你定義的分類標準自動歸類
- 輸出格式指定為結構化資料（JSON），方便後續處理
- 加入異常偵測：如果資料不完整，自動發通知

## 快速上手步驟

1. **安裝 n8n**：最簡單的方式是用 Docker，一行指令搞定
2. **取得 Claude API Key**：到 Anthropic Console 註冊並取得 API 金鑰
3. **建立第一個工作流**：在 n8n 介面中拖拉節點，連接觸發器和 Claude HTTP 節點
4. **測試與調整**：用測試資料跑幾次，調整 Prompt 到滿意為止
5. **啟用排程**：設定自動執行，讓工作流 24/7 運行

## 常見問題

**Q：n8n 要花錢嗎？**
自架版完全免費，雲端版有免費額度。Claude API 按使用量計費，小量使用每月約幾美元。

**Q：我完全不會寫程式，真的能用嗎？**
可以。n8n 是視覺化拖拉介面，Claude 的設定就是寫一段描述（Prompt）。整個過程不需要寫任何程式碼。

**Q：資料安全嗎？**
自架 n8n 的資料完全在你自己的伺服器上。Claude API 的資料不會被用於模型訓練。
