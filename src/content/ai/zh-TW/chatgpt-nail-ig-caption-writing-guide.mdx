「我做出了漂亮的作品，但就是不知道要怎麼配文字。」——這是無數美甲師的共同心聲。IG 演算法不只看圖片，文案的品質直接影響觸及率和互動率。但對於整天站著做美甲的師傅來說，要每天絞盡腦汁寫出吸引人的文案，實在是額外的負擔。

好消息是：ChatGPT 可以成為你的 24 小時文案助理。這篇文章提供針對美甲 IG 貼文設計的 Prompt 框架，以及真實可用的文案範例，讓你的社群行銷效率倍增。

## 為什麼美甲師需要專注在 IG 文案？

根據 2025 年社群媒體數據，Instagram 貼文中：
- 有好文案的貼文，互動率比純圖片高 **38%**
- 包含問句的文案，留言率比一般文案高 **2.5 倍**
- 加入適當 hashtag 的貼文，觸及率平均增加 **12%**

對美甲師而言，IG 不只是展示作品的地方，更是獲得新客和維繫老客的關鍵管道。好文案等於免費的廣告。

## ChatGPT 文案寫作的基本原則

在給 ChatGPT 任何 Prompt 之前，先了解幾個提升輸出品質的原則：

### 原則一：給它足夠的背景資訊

**差的 Prompt**：「幫我寫一篇美甲 IG 文案」

**好的 Prompt**：
```
幫我寫一篇台灣美甲工作室的 IG 文案。
作品：珊瑚粉凝膠甲，加入細金線幾何設計，圓形短甲
目標客群：25-35 歲台灣上班族女性
語氣：親切、有品味，帶點生活感
長度：100-150 字，包含 3-5 個 hashtag
```

### 原則二：指定語氣和風格

美甲工作室的 IG 文案有不同風格，告訴 ChatGPT 你的品牌個性：
- **親切日常型**：像在跟好朋友說話
- **專業知識型**：重視技術分享，建立專業形象
- **文青質感型**：文字有詩意，注重情境感
- **促銷行動型**：主打優惠、限時、緊迫感

### 原則三：要求多個版本

同一個 Prompt 請 ChatGPT 生成 3 個不同版本，再選最適合的。

## 30 個實用 Prompt 分類整理

### A 類：新作品發佈文（8 個）

**Prompt A1：日常款發佈**
```
我剛完成一組[顏色/設計描述]的美甲作品，客人是[客人職業/風格]。
幫我寫一篇 IG 貼文，著重於「這款設計適合什麼場合/什麼人」，
語氣親切，字數約 100 字，附 5 個相關 hashtag（台灣使用習慣）。
```

**Prompt A2：趨勢款發佈**
```
這是一款[設計風格]的指甲設計，靈感來自[靈感來源]。
請幫我寫一篇貼文，說明這個趨勢現在為什麼流行、
這款設計的特色是什麼，語氣時尚感強，適合對美甲有興趣的人。
字數 120-150 字，要有吸引人的開場白。
```

**Prompt A3：客人推薦款**
```
這是根據客人需求客製的一款設計：[設計描述]。
客人的需求是：[需求描述，例如要有貴氣感但不能太誇張]。
幫我寫一篇文案，說明如何理解客人需求並轉化成設計，
展現美甲師的專業和細心，語氣溫暖真誠。
```

**Prompt A4：作品故事型**
```
這款指甲設計叫做[設計名稱]，靈感是[靈感故事]。
請幫我用說故事的方式寫一篇 IG 文案，
讓人讀完會對這個設計有情感連結，不只是看一張美甲圖。
```

**Prompt A5：比較款**
```
今天完成了兩款截然不同的設計：一款是[設計A]，另一款是[設計B]。
請幫我寫一篇貼文，引導粉絲在留言選擇他們比較喜歡哪款，
增加互動率。語氣輕鬆好玩。
```

**Prompt A6：技術說明款**
```
這款作品用到了[技術名稱，例如暈染漸層/負空間/反法式]。
幫我寫一篇貼文，用簡單易懂的語言解釋這個技術是什麼、
難在哪裡，讓一般客人也能欣賞美甲師的專業功夫。
```

**Prompt A7：材料介紹款**
```
今天用到了[特殊材料，例如貓眼粉/雲母粉/乾燥花]做設計。
幫我介紹這個材料，包括：什麼是它、它的特色、
做出來的效果，語氣像在介紹一個有趣的發現。
```

**Prompt A8：前後對比款**
```
客人做了指甲修復/換款設計，前後對比很有看頭。
幫我寫一篇文案，描述[before]到[after]的轉變，
讓人感受到「美甲讓人煥然一新」的感覺，語氣正向激勵。
```

### B 類：促銷活動文（6 個）

**Prompt B1：新客優惠**
```
我們工作室推出新客體驗優惠：[優惠內容]，有效期限到[日期]。
請幫我寫一篇讓人想立刻預約的 IG 文案，
突出優惠的價值感，並製造一點預約緊迫感，加入 CTA（行動呼籲）。
```

**Prompt B2：節慶優惠**
```
[節日名稱]快到了，我們推出[節慶活動]。
幫我寫一篇結合節慶氣氛的文案，既有節日感又能推廣服務，
語氣溫暖節慶感強，字數 120-150 字。
```

**Prompt B3：回訪優惠**
```
針對三個月沒有來的老客人，我們推出回訪優惠：[優惠]。
請幫我寫一篇文案，用溫暖的語氣喚起老客人的美好回憶，
讓他們想回來，而不是只看到一個優惠。
```

**Prompt B4：候補名單文**
```
我們某個時段已滿，但想建立候補名單。
幫我寫一篇貼文，說明如何加入候補，
讓等待感覺像是一種「值得期待的特權」，而不是麻煩。
```

**Prompt B5：閒置時段填充文**
```
我們[星期幾][時段]有臨時空位，想快速找到客人。
幫我寫一篇有急迫感但不顯得狼狽的「今日限定」貼文，
讓客人覺得這是機會而不是剩餘空位。
```

**Prompt B6：漲價預告文**
```
我們計劃在[日期]後調整收費。
幫我以尊重、誠懇的語氣告知客人漲價消息，
說明漲價的原因，並感謝一直以來的支持，
讓客人接受這個消息而不感到反感。
```

### C 類：知識分享文（8 個）

**Prompt C1：保養知識**
```
幫我寫一篇美甲保養小知識的 IG 文案，主題是：[保養主題]。
用條列式整理 3-5 個重點，語氣像一個好朋友在分享有用資訊，
讓人讀完覺得學到東西，同時對美甲工作室的專業度留下印象。
```

**Prompt C2：破除迷思**
```
關於美甲，有一個常見的誤解是：[迷思內容]。
幫我寫一篇破解迷思的文案，先呈現迷思，再用專業的角度解釋正確觀念，
語氣教育性強但不說教，字數 120-150 字。
```

**Prompt C3：選色指南**
```
很多客人不知道怎麼選適合自己膚色的顏色。
幫我寫一篇簡單的選色指南文案，用易懂的方式說明[膚色類型]適合什麼顏色，
讓客人讀了之後更有信心選色，也更期待預約。
```

**Prompt C4：季節護理提醒**
```
現在是[季節]，美甲和手部護理需要特別注意[季節性問題]。
幫我寫一篇貼文，提醒客人這個季節的護理重點，
讓人感受到工作室真心關心客人，不只是在賣服務。
```

### D 類：互動增粉文（8 個）

**Prompt D1：投票問句**
```
幫我設計一個適合美甲 IG 的互動貼文，
用「你比較喜歡A還是B？」的格式，主題是：[選擇主題]，
讓粉絲在留言或 IG Story 投票，增加互動率。
```

**Prompt D2：問題徵集**
```
我想收集粉絲對美甲的疑問，作為之後內容的素材。
幫我寫一篇文案，邀請粉絲在留言提問，語氣親切，讓人有問問題的慾望。
```

**Prompt D3：個性測驗引流**
```
根據以下幾個關鍵字設計一個「你是哪種美甲風格？」的IG文案：
[風格A、風格B、風格C]，每種風格配合一個問題，
讓粉絲在留言回應，同時了解自己適合的風格。
```

## 進階技巧：建立你的個人文案風格

一直用 ChatGPT 生成文案，長期下來可能讓你的品牌失去個人特色。建議這樣使用：

1. **建立品牌語氣檔案**：把你覺得寫得好的文案存起來，告訴 ChatGPT「參考以下文案的語氣：[貼上範例]」
2. **只用 AI 打草稿，自己修飾**：AI 負責提供骨架，你加入個人溫度
3. **加入真實細節**：客人的真實反應、當天施作的小故事——這些 AI 不知道，但這些才是最有感染力的內容

ChatGPT 不是要取代你的聲音，而是要幫你把你想說的事說得更清楚、更有力量。學會用 AI 寫文案，不只省時間，更能讓你的工作室在眾多競爭者中建立清晰的品牌形象。

從今天開始，試著用一個 Prompt，讓 AI 幫你寫下一篇 IG 文案。你會發現，靈感其實從來都不缺，只是還沒找到說出來的方式。
