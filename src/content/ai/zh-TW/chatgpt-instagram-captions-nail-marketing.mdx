## 為什麼美甲師的IG文案常常寫到卡關？

你花了兩個小時做出一組超美的設計，拍了完美的照片，結果在手機上盯著「寫點什麼描述」的欄位發呆20分鐘。

這不是你不會寫，是**文案思維**跟**美甲技術**本來就是兩種完全不同的腦區。

ChatGPT 就是你那個永遠不會疲憊、24小時在線的文案夥伴。給它正確的指示，它能在30秒內生成你需要的文字——而你只需要根據自己的風格微調。

本篇給你10個真正有用的Prompt範本，以及讓IG文案有效提升互動率的策略。

---

## 寫IG文案前，先搞清楚「目的」

不同目的的貼文，需要不同的文案結構：

| 貼文目的 | 文案重點 |
|---|---|
| 展示作品 | 設計細節描述 + 情緒共鳴 + 預約CTA |
| 限時優惠 | 緊迫感 + 清楚說明優惠內容 + 行動呼籲 |
| 教育型內容 | 提供價值 + 引發討論 + 建立專業形象 |
| 互動型 | 提問 + 讓粉絲回覆 + 增加留言數 |

## 10個即用ChatGPT Prompt範本

### 1. 作品展示文案

```
我是台灣的美甲師，幫我寫一段IG貼文文案。
設計：漸層粉色加金箔，杏仁形凝膠，春季主題。
語氣：溫柔、有質感，像在說故事。
長度：100字左右，繁體中文。
最後加上預約的CTA，結尾加5個相關hashtag。
```

### 2. 季節限定優惠

```
幫我寫一段IG文案推廣夏季限定優惠：
優惠內容：7月預約凝膠服務，加送指甲保養油。
語氣：輕鬆活潑，有夏天的感覺。
加入緊迫感（限額20位）。
繁體中文，80字內。加3個hashtag。
```

### 3. 新款設計上市公告

```
我剛推出新款設計：星空鏡面漸層，價格$1200。
幫我寫一段IG發布文，讓粉絲感覺很期待、很想預約。
語氣：帶一點神秘感和興奮感。
繁體中文，100字。加5個hashtag，hashtag要包含「鏡面美甲」和「星空指甲」。
```

### 4. 互動型貼文（提問）

```
幫我想一個適合美甲師IG的互動型貼文問題。
主題：讓粉絲選擇他們喜歡的款式（A vs B）。
設計A：霧面奶茶色，簡約風。
設計B：亮面紅色，耶誕節風格。
語氣活潑，鼓勵留言互動。繁體中文，50字內。
```

### 5. 客戶見證轉PO文

```
客戶說：「做完真的超美，已經被朋友問了三次在哪裡做的！」
幫我把這段評語改寫成IG貼文文案，展現客戶的喜悅和品牌的專業感。
加上感謝客戶的語氣，並邀請其他人也來體驗。
繁體中文，80字內。
```

### 6. 限時動態（Stories）腳本

```
幫我設計一個美甲沙龍IG限時動態的3頁腳本：
主題：介紹我的「春季手繪花卉」新系列。
每頁一句話或一個視覺重點描述（文字要放在畫面上的）。
語氣：清新，有視覺感，像雜誌風格。繁體中文。
```

### 7. 保養知識型貼文

```
幫我寫一篇美甲保養知識的IG文案。
主題：凝膠指甲的正確保養方式，避免翹起剝落。
格式：用3個重點條列，每點15字內。
最後加一句引導粉絲儲存這篇的CTA（例如：收藏起來備用！）。
繁體中文，加5個教育型hashtag。
```

### 8. 週末空檔填單

```
我明天下午2點到4點有2個臨時空檔，幫我寫一段IG文案請粉絲搶預約。
語氣：輕鬆、有點稀缺感，但不要硬推銷。
提醒可以直接回覆私訊「我要」來預約。
繁體中文，50字內。
```

### 9. 節慶行銷文案

```
情人節快到了，幫我寫一段適合美甲沙龍IG的情人節主題貼文。
可以提到：送自己一份禮物、情侶互相推薦、限定情人節款式。
語氣：溫暖浪漫，不要太商業感。
繁體中文，100字。加5個情人節美甲hashtag。
```

### 10. 關注成長型文案（Reels/連結貼文）

```
我剛發布了一部美甲教學Reels，教如何自己做簡單的暈染漸層。
幫我寫一段文案引導粉絲去看影片：
- 說明影片教了什麼（30秒就能學會）
- 引導他們追蹤帳號獲得更多教學
- 語氣：友善親切，像朋友分享
繁體中文，80字。加5個hashtag。
```

## Hashtag 策略：AI幫你找對標籤

直接問ChatGPT：

```
幫我列出30個適合台灣美甲師IG使用的hashtag，
分成三類：
1. 高曝光大標籤（100萬以上貼文）
2. 中型精準標籤（10萬到100萬貼文）
3. 小眾利基標籤（1萬以下，互動率高）
繁體中文和英文各一份。
```

## 讓文案更有效的三個訣竅

### 1. 加入「你」的視角

ChatGPT生成的初稿是通用版。加入你自己的語氣、你沙龍的個性，讀起來才真的像你。

### 2. 第一句話最重要

IG只顯示前兩行，第一句話決定粉絲要不要展開看全文。讓ChatGPT重寫第一句，直到它讓你想繼續讀。

### 3. 固定一個行動呼籲（CTA）

每篇貼文只要一個CTA：要嗎預約、要嗎留言、要嗎儲存。不要同時問三件事。
