## 為什麼美甲店需要 LINE Bot 客服？

台灣美甲師最常遇到的場景：

- 正在幫客人做凝膠美甲，手機一直響
- 每天都在回一模一樣的問題：「有沒有空檔？」「做凝膠多少錢？」「需要預約嗎？」
- 晚上想休息，但怕漏掉潛在客人訊息

LINE Bot 能解決這三個問題：自動回覆常見問題、24 小時接收預約意願、讓你在工作中不被打斷。

這篇教你從零開始設定，不需要工程師，不需要寫程式。

---

## 基礎架設：LINE Official Account

### 第一步：申請 LINE 官方帳號

1. 前往 [LINE for Business](https://www.linebiz.com/tw/) 申請 LINE Official Account（LINE OA）
2. 選擇「一般帳號」（免費版）開始
3. 填入店名、類別（美容/美甲）、聯絡資料

免費版每月可發送 **200 則群發訊息**，自動回覆則**無限制**，對大多數小型美甲工作室已足夠。

### 第二步：設定基本自動回覆

在 LINE OA Manager 後台：
1. 進入「自動回覆訊息」
2. 新增以下常見問題的關鍵字回覆：

| 關鍵字 | 自動回覆內容 |
|--------|------------|
| 價格、費用、多少錢 | 「凝膠美甲單色 $800 起，彩繪款式 $1200 起，詳細價格表請點此連結：[連結]」 |
| 預約、空檔、有沒有位置 | 「歡迎預約！請告知您希望的日期和時間，我會盡快確認。也可以直接使用線上預約：[連結]」 |
| 地址、位置、怎麼去 | 「我們位於[地址]，捷運[站名]出口步行5分鐘，詳細地圖：[Google Maps 連結]」 |
| 時間、營業 | 「營業時間：週二至週日 10:00–20:00，週一公休。」 |

這幾個關鍵字涵蓋了 **80% 的常見詢問**，設定完後立即生效。

## 進階功能：讓回覆更聰明

### 用 ChatGPT 寫更自然的回覆文案

LINE OA 的自動回覆預設很制式，可以請 ChatGPT 幫你潤稿：

> **「我是台灣獨立美甲師，請幫我把以下制式回覆改得更溫暖自然：『我們的凝膠美甲費用如下……』」**

讓 AI 幫你的機器人說話說起來更像真人，增加客人好感。

### 設定「快捷選單」（Rich Menu）

Rich Menu 是 LINE 聊天畫面下方的互動選單，可以設計成：

```
┌─────────────────────────────┐
│  📋 查看價格  │  📅 線上預約  │
│──────────────┼──────────────│
│  📍 店家位置  │  💬 聯絡我們  │
└─────────────────────────────┘
```

設定方式：LINE OA Manager → 圖文選單 → 新增

這樣客人一進對話框，就能快速找到他們需要的資訊，不用等你回覆。

### 導入第三方 AI 客服（進階）

如果想要更智慧的對話（例如 AI 能理解「我想做春天感的凝膠，你有什麼推薦？」），可以考慮：

- **ManyChat**：LINE + Instagram 整合，有基本 AI 對話功能
- **Landbot**：拖拉式對話流程設計，適合複雜的預約流程
- **n8n + ChatGPT API**：技術門檻較高，但完全客製化，可以做到真正的 AI 對話

## 預約流程整合：讓 LINE Bot 直接幫你收預約

最理想的設定：
1. 客人發訊息詢問空檔
2. LINE Bot 自動回覆「請問您希望預約哪天？」
3. 收到客人回覆後，發送 Google 表單或 Calendly 連結
4. 客人填表確認，系統自動寄出確認訊息

這個流程你可以用 **Google Forms + LINE OA 自動回覆** 在 30 分鐘內建立完成。

## 常見問題

**Q：LINE Bot 需要付費嗎？**
LINE OA 基本功能免費，自動回覆無限制。進階功能（如 API 整合）需要申請 Messaging API 帳號（免費申請，但需要技術能力）。

**Q：客人會不會覺得跟機器人說話很奇怪？**
只要回覆文案寫得自然，大多數客人不會在意。關鍵是：機器人能快速給答案，客人體驗比等你手動回更好。

**Q：我需要工程師嗎？**
基本的關鍵字自動回覆完全不需要。Rich Menu 設計需要稍微懂得圖片尺寸。進階 AI 整合才需要技術協助。
