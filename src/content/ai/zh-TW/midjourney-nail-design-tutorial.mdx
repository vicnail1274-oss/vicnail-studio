## Midjourney 是什麼？需要什麼才能用？

Midjourney 是一個透過文字描述生成高品質圖像的 AI 工具。你輸入一段描述（Prompt），幾秒鐘後它就吐出 4 張不同風格的圖給你選。

**需要準備的東西：**
- 一個 Discord 帳號（免費）
- Midjourney 訂閱（基本方案約 10 美金/月）
- 用英文寫 Prompt（Midjourney 英文指令效果最好）

雖然需要付費，但對美甲師來說，一個月幾百塊換來可以無限生成甲款參考圖，CP 值非常高。

## 核心技巧：寫出好看甲款的 Prompt 結構

Prompt 寫法直接決定輸出圖的品質。以下是一個高效的結構公式：

```
[甲款風格] nail art, [顏色描述], [材質/效果], [細節描述],
close-up macro photography, high detail, white background
```

### 實際範例對比

**差的 Prompt：**
```
beautiful nail art
```
→ 生成的圖很普通，什麼都有一點，什麼都不出色

**好的 Prompt：**
```
minimalist nail art, soft beige and dusty rose, matte finish,
thin gold line details, almond shape, close-up macro photography,
high detail, white background
```
→ 生成的圖有明確風格、顏色和細節

## 進階技巧：讓圖片更接近真實甲款

### 技巧 1：加上指甲形狀關鍵字

Midjourney 對指甲形狀很敏感，加入形狀描述能讓圖更精準：
- `almond shape`（杏仁形）
- `coffin shape`（棺材形）
- `oval shape`（橢圓形）
- `square shape`（方形）
- `stiletto shape`（尖形）

### 技巧 2：指定材質與效果

- `matte finish`（霧面）
- `glossy gel finish`（光澤凝膠）
- `chrome mirror effect`（鏡面感）
- `cat eye effect`（貓眼）
- `glitter texture`（閃粉質感）

### 技巧 3：加上攝影參數讓圖更真實

在 Prompt 末尾加上：
```
close-up macro photography, 85mm lens, professional studio lighting,
high resolution, photorealistic
```

這樣生成的圖會更像真實美甲照片，而不是插畫風格。

### 技巧 4：使用 `--ar` 設定比例

美甲圖片用 `--ar 1:1` 或 `--ar 4:3` 很合適，方便 IG 或 Pinterest 發文：
```
[你的 Prompt] --ar 1:1
```

## 美甲師如何把 AI 生成圖用在工作中？

### 用途 1：建立自己的作品集風格板

還沒有真實作品的新手美甲師，可以先用 Midjourney 生成各種風格的圖，建立視覺風格板，讓客人了解你擅長哪些方向。

**注意**：在展示時標明「AI 生成靈感圖」，不要假裝是自己的真實作品。

### 用途 2：客人諮詢前的視覺確認

客人預約時描述想要的款式，你用 AI 先生成接近的圖給她確認，避免「做出來和想像的不一樣」的情況。這個流程可以大幅降低後期修改。

### 用途 3：季節甲款的靈感庫

每季換季前，用 Midjourney 生成 20-30 張當季流行風格的甲款圖，貼在 IG 或 LINE 官帳，讓客人投票選最想做的款式。這是免費的市場調查。

## 常見問題

**Q：Midjourney 生成的圖可以商業使用嗎？**
A：訂閱付費方案的用戶通常有商業使用權，但建議查閱最新的 Midjourney 使用條款確認。

**Q：中文 Prompt 有效嗎？**
A：Midjourney 可以理解中文，但效果通常比英文差。建議重要的 Prompt 用英文寫，或用 ChatGPT 幫你把中文描述翻譯成英文 Prompt。

**Q：生成的圖跟我想要的差很多，怎麼辦？**
A：試試在原 Prompt 加上 `--no [不要的元素]` 排除不想要的部分，並且更具體地描述細節。
