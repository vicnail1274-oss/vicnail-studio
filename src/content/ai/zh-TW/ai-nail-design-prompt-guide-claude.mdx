## 用 Claude AI 生成夢幻美甲設計：完整提示詞指南

客人坐下來的第一句話常常是：「我不知道要做什麼，你幫我想。」

以前這要靠美甲師的經驗和腦袋。現在，你有 Claude AI 當隊友。這篇文章會教你怎麼用對提示詞，讓 AI 在 30 秒內給你 5-10 個具體可執行的設計方案。

---

## 為什麼選 Claude？

AI 工具很多，為什麼推薦從 Claude 開始？

| 工具 | 優點 | 美甲應用場景 |
|------|------|------------|
| **Claude** | 理解中文語境強、描述細緻、可追問 | 設計發想、客戶溝通稿、說明文案 |
| **Midjourney** | 直接生成圖片 | 視覺樣板、色卡展示 |
| **ChatGPT** | 通用性強 | 行銷文案、IG 貼文 |

**簡單說：** Claude 擅長「用文字描述設計」，Midjourney 擅長「直接畫出來」。這兩個搭配使用是最強組合——先用 Claude 想好設計概念，再餵給 Midjourney 出圖。

## 5 大類即用提示詞模板

### 一、季節配色類（最常用）

```
我需要 5 個適合台灣 [季節] 的凝膠美甲設計。
要求：
- 顏色：[色系需求，例如：清爽系/大地色/鮮豔]
- 風格：[例如：簡約/可愛/成熟]
- 特別需求：[例如：上班族適用/約會款/日系風]
請給每個設計的名稱、主色+輔色、建議裝飾元素、適合哪類客人。
```

**實際範例輸出（春季）：**

Claude 回覆會包含：「嫩芽奶茶」（主色：米白+抹茶綠，裝飾：小花貼片）、「粉霧漸層」（主色：薰衣草+白，裝飾：月暈設計）等具體方案。

---

### 二、主題活動類

```
客人說要去 [活動/場合]，幫我想 3 個適合的凝膠美甲設計。
活動：[婚禮賓客/日本旅遊/公司聚餐/演唱會]
客人偏好：[喜歡/不喜歡的元素]
預算等級：[基本款/中階/高端精品]
每個設計請說明：款式名稱、顏色搭配、裝飾重點、施作時間約幾分鐘。
```

---

### 三、客人風格分析類（溝通神器）

```
幫我分析這位客人適合什麼美甲風格：
年齡層：[20-25 / 25-35 / 35+]
職業：[學生/白領/服務業/自由業]
常穿衣服顏色：[中性色 / 鮮豔 / 黑白灰]
個性描述：[活潑 / 低調 / 時尚感強]

請推薦 3 個方向，每個方向給 2 個具體設計。
```

這個提示詞特別適合用在：客人說「你幫我決定就好」的時候，快速給出專業建議。

---

### 四、修改優化類（已有設計，想升級）

```
我有一個基本款設計：[描述現有設計，例如：純色奶茶色 + 一根指甲貼片]
客人說她想要「更特別一點，但不要太誇張」
請幫我提供 3 個升級方向：
1. 加什麼裝飾
2. 要用什麼顏色搭配
3. 和原本設計比，費用大概提高多少（用比例表示）
```

---

### 五、IG 貼文文案類（行銷必備）

```
幫我寫一則 IG 貼文，介紹這款凝膠美甲：
設計名稱：[設計名稱]
特色：[2-3 個特色]
適合族群：[目標客群]

要求：
- 繁體中文
- 加 5 個主題標籤（#hashtag）
- 結尾加預約 CTA
- 語氣：[活潑/質感/親切]
- 字數：100-150 字
```

## 進階技巧：追問讓答案更好

Claude 最強的地方是**可以對話**。第一輪答案不滿意，直接追問：

- 「第 3 個設計改成更適合秋天的顏色」
- 「用台灣客人習慣的說法重新描述」
- 「幫我把這個設計改成適合短指甲的版本」

每次追問，Claude 都會根據你的反饋調整，越問越精準。

---

## 實際工作流程（5 分鐘用起來）

1. 客人說明需求（1 分鐘）
2. 把需求貼入 Claude 提示詞模板（30 秒）
3. 看 Claude 的回覆，選出 2-3 個方向給客人選（1 分鐘）
4. 客人選好後，把設計描述餵給 Midjourney 出參考圖（1-2 分鐘）
5. 確認設計、開工

從「不知道要做什麼」到「設計確認」，5 分鐘搞定。
