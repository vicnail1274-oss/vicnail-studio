## 什麼是Stable Diffusion

Stable Diffusion是一個開源的AI圖像生成模型，由Stability AI在2022年發布。與Midjourney（需要訂閱）不同，Stable Diffusion可以完全免費在本地電腦運行，也不需要網路連線（本地版本）。

### 主要版本比較

| 版本 | 特點 | 適合用途 |
|------|------|----------|
| SDXL | 高解析度，細節豐富 | 精細的美甲藝術圖 |
| SD 1.5 | 輕量，速度快，生態豐富 | 快速生成大量靈感圖 |
| SD 3.0（2026新版） | 最高品質，理解力最強 | 最接近照片的效果 |

### 使用方式選擇

**本地安裝（ComfyUI或AUTOMATIC1111）**：
- 優點：完全免費，無限生成，資料不上傳雲端
- 缺點：需要有顯示卡（至少6GB VRAM），安裝設置有技術門檻

**雲端平台（Civitai Generate、Leonardo.ai等）**：
- 優點：無需安裝，立刻可用
- 缺點：有使用次數/費用限制

**建議**：如果你有電腦顯卡，值得投入時間安裝本地版；如果你想先試試效果，先用免費的雲端平台。

## 美甲設計的核心Prompt技巧

Prompt（提示詞）是決定AI生成效果的最關鍵因素。以下是針對美甲設計的Prompt編寫指南。

### 美甲Prompt的基本結構

```
[主題/風格] + [顏色] + [甲型] + [細節元素] + [拍攝風格] + [品質關鍵詞]
```

### 範例Prompt（英文效果最好）

**日系小花甲**：
```
Japanese nail art, floral gel nails, soft pink and white,
sakura flowers, delicate design, short oval nails,
close-up macro photography, studio lighting,
masterpiece, best quality, highly detailed
```

**極光貓眼**：
```
aurora chrome nails, cat eye effect, deep blue purple shift,
oval medium length nails, glossy finish,
professional nail salon photography, studio light,
8k quality, photorealistic
```

**法式美甲進階版**：
```
elegant French manicure, thin gold line tips,
nude pink base, almond shaped nails,
luxury nail salon photography, macro lens,
natural lighting, minimalist aesthetic, high resolution
```

**秋冬深色系**：
```
dark burgundy gel nails, velvet matte texture,
gold foil accents, squoval shape,
autumn aesthetic, close-up product shot,
professional lighting, rich colors
```

### 負面Prompt（告訴AI不要生成什麼）

加入Negative Prompt可以大幅提升品質：

```
ugly, deformed, blurry, bad anatomy, extra fingers,
missing nails, low quality, watermark, text, logo
```

## 適合美甲設計的Stable Diffusion模型

標準的Stable Diffusion模型對真實指甲圖片的生成效果一般，但社群上有很多針對特定風格微調的LoRA（微調模型）：

### 推薦的美甲相關LoRA

**在Civitai（civitai.com）搜尋以下關鍵詞找到社群分享的模型**：
- "nail art" LoRA
- "Japanese nail" LoRA
- "gel nail" LoRA

**使用方式**：
在Prompt中加入 `<lora:模型名稱:0.8>` 即可啟用LoRA（強度0.8為一般推薦值）

### 基底模型選擇

**Realistic Vision（寫實風格）**：適合生成接近真實照片的指甲圖
**DreamShaper（夢幻風格）**：適合生成藝術感強的指甲插畫
**CounterfeitXL（日系風格）**：日系美甲效果極佳

## ControlNet：讓AI按照你的甲型生成

ControlNet是Stable Diffusion最強大的功能之一：讓你提供一張「結構圖」，AI生成的圖片會遵循這個結構，只改變風格和顏色。

### 美甲ControlNet應用場景

**場景一：按照客人指甲形狀生成設計**

1. 拍一張客人現有指甲的照片
2. 用ControlNet的Canny（邊緣檢測）處理這張照片
3. 輸入你想要的設計Prompt
4. AI生成的圖片會保持相同的指甲形狀，但改變設計

這讓你可以直接向客人展示「你的指甲形狀做了這個設計會是什麼效果」。

**場景二：参考構圖生成不同配色**

找到一個你喜歡的構圖，用ControlNet鎖定構圖，然後嘗試10種不同配色方案——在30秒內完成傳統需要數小時的配色測試。

## 實際工作流程：美甲設計發想

### 流程一：客人諮詢現場輔助

**情境**：客人到店說「我想要日系風，最近看到一款有點透明感的，帶點粉色，偶爾加一點金色點綴」

**傳統流程**：翻閱樣本書 + 手機搜圖 → 花費5-10分鐘 → 有時候找不到理想的

**AI輔助流程**：
1. 在ComfyUI或雲端平台輸入：
   ```
   Japanese nail art, sheer milky pink gel nails,
   translucent jelly texture, gold foil accents,
   oval shaped, natural light photography
   ```
2. 生成4-6張圖（約30秒）
3. 請客人從中選擇最接近的風格
4. 根據客人反饋調整Prompt，再生成一輪
5. 確認方向後開始施作

**效果**：設計確認時間從10分鐘縮短至2-3分鐘，且客人對最終效果的滿意度更高（因為有視覺確認）。

### 流程二：社群貼文靈感預備

每次要為IG貼文準備美甲設計靈感，預先用AI生成一批靈感圖庫：

1. 設定10個當季熱門關鍵詞（例如：珊瑚、蝴蝶結、極光）
2. 每個關鍵詞生成5-10張靈感圖
3. 篩選品質好的存入靈感庫
4. 製作社群貼文時直接從靈感庫取用

### 流程三：客製化禮品券或品牌素材

用特定的Prompt生成符合你品牌調性的插畫圖，用於：
- 禮品券背景圖
- 打卡牆設計
- 包裝素材

## 常見問題

**Q：Stable Diffusion可以生成準確的真實照片嗎？**
使用寫實型基底模型（如Realistic Vision）加上好的Prompt，可以生成非常接近真實照片的圖像，但仍不完美——手指和指甲的細節有時會出現不自然的扭曲，需要從多個生成結果中挑選。

**Q：我沒有顯示卡，可以用CPU運行嗎？**
可以，但生成一張圖可能需要5-10分鐘（vs 顯示卡的5-10秒）。建議使用雲端平台更為實際。

**Q：Midjourney和Stable Diffusion哪個效果更好？**
對於一般美甲靈感圖，Midjourney的美化效果通常更好且更一致，但需要訂閱費（約10-12美元/月）。Stable Diffusion提供更多控制彈性，適合進階用戶。

**Q：生成的圖可以直接印出來給客人看嗎？**
可以，AI生成的靈感圖用於客人諮詢和確認設計方向是完全合理的應用場景。
