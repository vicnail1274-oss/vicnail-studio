## AI 可以幫美甲業做什麼？比你想像的多

很多人用 ChatGPT 只是偶爾問問天氣、翻個字，覺得「好像也還好」。但如果你是美甲師，或是美甲愛好者，ChatGPT 可以做的事遠不止這些。

這篇整理了 10 個具體的使用場景，每個都附上可以直接用的 Prompt 範例。你不需要是科技達人，只需要知道怎麼問對問題。

---

## 使用前的準備：讓 ChatGPT 更了解你

在開始之前，給 ChatGPT 一個背景設定可以讓後面所有的對話都更準確：

```
我是一個在台灣開美甲工作室的老闆，主要客群是 25-40 歲女性，
偏好日系精緻風和法式甲，工作室以個人化服務和高品質見長。
```

這段設定可以讓 ChatGPT 在後面所有的對話中，都用對應的背景給你建議。

---

## 場景 1：選色推薦 — 幫客人找到最適合的甲色

**使用情境**：客人說「想換個新顏色，但不確定自己適合什麼」。

**Prompt 範例：**
```
我的客人膚色偏暖黃、中等深淺，她通常偏好低調的日系風格，
不喜歡太誇張的設計。現在是春天，她想嘗試新的顏色。
請給我 5 個推薦色系，每個說明為什麼適合她，
以及可以搭配的設計風格。
```

ChatGPT 可以給出有邏輯依據的建議，不是隨便說「粉色很百搭」這種。

---

## 場景 2：設計靈感 — 客人說不清楚想要什麼

**使用情境**：客人只說「我想要很特別的，但又不要太誇張」。

**Prompt 範例：**
```
客人想要「有設計感但不誇張」的美甲，可以上班穿，
指甲長度中等，現在是秋季。
請給我 5 個具體的甲款設計方向，每個用 3 句話描述清楚，
包含顏色、設計元素和適合的場合。
```

這樣你可以把 ChatGPT 給的描述作為諮詢的起點，讓客人從中挑選方向。

---

## 場景 3：IG 文案寫作

**使用情境**：每週要發 IG 貼文但不知道寫什麼。

**Prompt 範例：**
```
我剛完成了一個客人的法式漸層甲款，顏色是奶白色到粉裸色。
請幫我寫一篇 IG 貼文，風格要溫暖親切，包含：
1. 引起共鳴的開場（關於穿搭或日常）
2. 對這個甲款的描述（2-3 句）
3. 呼籲行動（歡迎預約或私訊）
4. 5 個繁體中文 hashtag
字數控制在 150 字以內。
```

## 場景 4：客戶詢問回覆範本

**使用情境**：常常收到一樣的問題，回覆要花很多時間。

**Prompt 範例：**
```
請幫我寫 3 個不同長度的回覆範本，用於當客人問「延甲和光療的差別是什麼？」
1. 簡短版（LINE 快速回覆用）：50 字以內
2. 中等版（詳細說明用）：150 字
3. 完整版（網站 FAQ 用）：300 字
語氣要親切專業，適合台灣的美甲客群。
```

## 場景 5：促銷活動企劃

**使用情境**：母親節快到了，想做一個促銷活動但不知道怎麼規劃。

**Prompt 範例：**
```
我想為我的美甲工作室規劃一個母親節促銷活動（5 月第二個週日）。
工作室主要服務：光療甲 1200 元、延甲 1800 元、手部護理 600 元。
客群：25-45 歲女性。
請幫我設計 3 個不同的促銷方案，每個包含：
- 活動名稱
- 優惠內容
- 適合的宣傳文案（50 字）
- 這個方案的優點和適用情境
```

## 場景 6：保養建議給客人

**使用情境**：客人問甲片保養問題，你想給出完整的書面建議。

**Prompt 範例：**
```
客人剛做完光療延甲，問我如何保養讓甲片維持更久。
請幫我寫一份完整的保養指南，包含：
- 日常護理（每天該做的事）
- 應該避免的行為
- 產品推薦（不要指定品牌，用類型代替）
- 什麼情況需要回來補修
格式：清單形式，方便客人截圖保存。
```

## 場景 7：價目表優化

**使用情境**：想重新整理服務價目，讓它更有說服力。

**Prompt 範例：**
```
我的美甲工作室現有服務：
- 基本光療 800 元
- 光療延甲 1500 元
- 手部護理 500 元
- 卸甲 200 元
請幫我重新包裝這些服務的名稱和描述，讓它們聽起來更有質感、
更值得這個價格，可以放在官網和 IG bio 上。
```

## 場景 8：處理負面評論

**使用情境**：收到一則讓你不知道怎麼回應的差評。

**Prompt 範例：**
```
我收到這樣一則 Google 評論：「做完一週就翹邊了，很失望。」
請幫我寫一個專業而有溫度的回覆，要：
1. 表達歉意但不承認技術有問題（因為可能是客人保養問題）
2. 說明我們的補修政策
3. 邀請客人回來處理
4. 讓其他看到的人覺得這是一家負責任的工作室
```

## 場景 9：新客戶開發訊息

**使用情境**：想發一個訊息給之前詢問過但沒有預約的潛在客戶。

**Prompt 範例：**
```
有一個潛在客人上個月私訊詢問過我的服務，但最後沒有預約。
請幫我寫一則不打擾、不催促的友善跟進訊息，
可以在她的 IG 或 LINE 上發送。
包含：問候、新的服務亮點、輕鬆的呼籲行動。
字數：80 字以內。
```

## 場景 10：月結報告分析（給自己看的）

**使用情境**：想了解上個月的業績狀況，找出可以改善的地方。

**Prompt 範例：**
```
這是我上個月的服務資料：
- 總服務次數：45 次
- 新客：12 人，舊客：33 人
- 最熱門服務：光療延甲（20 次）
- 次熱門：基本光療（15 次）
- 手部護理只有 3 次
- 平均客單價：1350 元
請幫我分析：1) 現在的業績結構有什麼問題？
2) 手部護理銷量低可能的原因？
3) 提高客單價的 3 個具體建議？
```

## 讓 ChatGPT 更好用的 3 個技巧

### 技巧 1：給它角色設定

在 Prompt 前面加：「你是一位擁有 10 年經驗的台灣美甲業行銷顧問」，它的建議品質會明顯提升。

### 技巧 2：要求具體格式

「請用清單格式」、「每個建議不超過 2 句話」——越具體的格式要求，越容易直接使用輸出內容。

### 技巧 3：叫它扮演你的客戶

「你現在是一個第一次想做延甲、但有點緊張的客人，請問我有哪些顧慮？」——這樣可以幫你發現客戶諮詢中你可能沒有回答到的問題。

---

## ChatGPT 的限制：不要過度依賴

- **它不了解你的客人**：它給的建議是通用的，你需要根據具體客人做調整
- **它可能給出過時資訊**：美業趨勢變化快，記得驗證 AI 給的流行色或市場資訊
- **語氣要調整**：AI 輸出的語氣通常比較正式，記得加入你自己的口吻再發布
