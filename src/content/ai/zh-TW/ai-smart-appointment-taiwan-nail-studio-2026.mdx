## 你還在手動回每一則預約訊息嗎？

台灣美甲工作室的預約幾乎都靠 LINE，但問題在於——90% 的預約訊息都是重複的：

「請問週六有空嗎？」
「做凝膠甲要多久？」
「可以做延甲嗎？請問費用？」

每一則都要親自回覆，一天累積下來可能花掉你 1–2 小時的行政時間。這些時間完全可以讓 AI 幫你處理。

---

## 智能預約系統的核心架構

一套完整的台灣美甲工作室 AI 預約流程，通常由三個層次組成：

### 第一層：LINE Bot 自動問答

客戶透過 LINE 官方帳號發訊息 → LINE Bot 自動辨識問題類型 → 回覆常見問題或引導進入預約流程。

常見設定場景：
- 關鍵字觸發：「預約」「空檔」「費用」「做甲」→ 自動回覆對應模板
- 菜單式按鈕：客戶點選「查看服務項目」「立即預約」「聯絡美甲師」
- 營業時間外：自動回覆「目前非服務時間，您的訊息已收到，將於 XX 前回覆」

### 第二層：線上預約表單（整合 LINE）

LINE Bot 在適當時機把客戶導向預約表單，收集：
- 服務項目（凝膠甲/延甲/凝膠保養/卸甲）
- 偏好日期與時段（3個選項）
- 手況說明（如指甲長度、是否有殘留舊甲膠）
- 參考圖片上傳（減少現場溝通）

推薦工具：**Calendly**（支援 LINE 連結）、**Setmore**（免費版）、或自建的 Google Forms + n8n 自動化流程

### 第三層：自動確認與提醒

預約成功後，系統自動：
1. 發送確認訊息（含日期、時間、服務項目摘要）
2. 預約前 24 小時發送提醒
3. 預約前 2 小時發送最後確認
4. 服務完成後 3 天發送回訪邀請

## 實際設定教學：LINE Bot + Calendly + n8n

### 前置準備

- LINE 官方帳號（免費版即可）
- Calendly 帳號（免費版支援基本預約）
- n8n（可用免費雲端版，或自架）

### Step 1：LINE Bot 關鍵字自動回覆

在 LINE 官方帳號後台 →「自動回覆訊息」→「關鍵字回覆」

設定以下關鍵字觸發組：

| 關鍵字 | 自動回覆內容 |
|--------|------------|
| 預約、約、空檔 | 「你好！點此連結查看可預約時段：[Calendly 連結]」 |
| 費用、收費、多少錢 | [費用表圖片 + 加入好友優惠說明] |
| 做多久、時間 | 「凝膠甲約 60–90 分鐘，延甲約 90–120 分鐘，建議預留充裕時間哦！」 |
| 卸甲、卸掉 | 「卸甲服務 $XXX，建議提前預約以確保時間充足。點此預約：[連結]」 |

### Step 2：Calendly 預約頁設定

1. 建立一個「美甲預約」事件類型
2. 設定服務時長（依項目）
3. 開啟「收集額外資訊」欄位：服務項目、手況說明、參考圖片連結
4. 自訂確認頁文字：「預約成功！將於 XX 前確認。請準時到場哦！」

### Step 3：n8n 串接自動提醒

在 n8n 建立以下工作流：

```
Calendly 新預約觸發
  → 等待至預約前 24 小時
  → LINE Bot 發送提醒訊息
  → 等待至預約前 2 小時
  → LINE Bot 發送最後確認
  → 預約時間後 3 天
  → LINE Bot 發送回訪邀請
```

這個工作流一旦設定完成，每次預約都會自動走完整個流程，完全不需要手動操作。

## 更進一步：AI 問診問卷整合

在預約表單中加入智能問診問卷，可以讓美甲師提前了解客戶狀況：

**問診問卷建議欄位：**
- 指甲目前狀態（自然甲/有舊凝膠甲/已延甲）
- 指甲長度偏好
- 是否有過敏史或指甲問題
- 這次想做的風格（提供圖片選項）

透過 Typeform 或 Google Forms 建立問卷，並在預約確認後自動寄送，客戶填完後資料直接匯入你的預約管理後台。

## 常見問題與解決方案

**Q：LINE Bot 自動回覆有沒有字數限制？**
有，單則訊息上限 500 字。建議重要資訊分成多則，或用圖片卡片呈現費用表。

**Q：客戶不習慣用 Calendly，怎麼辦？**
可以在 Calendly 連結旁加上一句「也可直接回訊，我幫你安排」，保留人工備案入口，讓不習慣線上預約的客戶仍有選擇。

**Q：如果有客戶反悔或改期，系統如何處理？**
Calendly 支援客戶自助改期與取消，改期後 n8n 會自動觸發新的提醒流程。
