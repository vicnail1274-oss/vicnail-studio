## 什麼是凝膠指甲？

凝膠指甲是使用特殊的凝膠材料，透過 UV 或 LED 燈照射固化，形成堅固且光澤的指甲塗層。相比傳統指甲油，凝膠指甲更持久（可維持 2–4 週）、更有光澤，且不易因日常摩擦而掉色。

對美甲初學者來說，了解凝膠的基礎知識，可以幫助你在沙龍溝通時更有信心，也更清楚自己想要什麼效果、適合哪種凝膠類型。

## 凝膠的種類

### 1. 硬式凝膠（Hard Gel）

硬式凝膠固化後非常堅硬，適合用來延長指甲長度或建構指形。因為卸除時需要使用電動銼刀或手工銼磨，**無法用浸泡卸甲液的方式去除**，是三種凝膠中最耐用的一種。

**適用情境：**
- 指甲本身很短，想做延甲造型
- 需要高強度的日常耐用性
- 做精緻浮雕設計的底層基礎

### 2. 軟式凝膠（Soft Gel / Soak-off Gel）

最常見的沙龍凝膠種類。固化後有一定的彈性，**可以用含丙酮的卸甲液浸泡 10–15 分鐘輕鬆去除**，對自然甲的傷害較低。幾乎所有凝膠色膠都是軟式凝膠配方。

**適用情境：**
- 一般沙龍美甲（單色、漸層、圖案）
- 希望定期更換顏色的客人
- 自然甲保養為主，不想傷甲

### 3. 建構凝膠（Builder Gel）

介於硬式和軟式之間，兼具一定硬度與可浸泡卸除的特性。主要用於**加強甲板厚度**、修補斷裂或扁平的指甲，也可以用來做薄延甲。

**適用情境：**
- 指甲容易斷裂或太薄的人
- 想讓指形更飽滿但不需要長延甲
- 修補指甲斷裂的緊急處理

## 基礎工具清單

開始做凝膠美甲前，以下工具缺一不可：

| 工具 | 說明 |
|------|------|
| UV / LED 凝膠燈 | 固化凝膠的核心設備。LED 燈固化快（30–60 秒），UV 燈固化慢（120 秒）但相容性更廣 |
| 底膠（Base Coat） | 增加凝膠與自然甲的附著力，同時保護甲面 |
| 色膠（Color Gel） | 提供顏色，通常需塗 2–3 層 |
| 封層（Top Coat） | 保護色彩、增加光澤，可選擇亮面或霧面 |
| 甲面處理液（Nail Prep） | 去除油脂和水分，讓凝膠附著更穩 |
| 清潔液（Cleanser） | 照燈後擦除未固化的殘留層 |
| 砂條 / 砂棒 | 輕微打磨甲面，提升附著力 |
| 橙木棒 / 推皮棒 | 整理甲周皮膚，避免凝膠沾到皮膚 |

## 標準上色步驟

了解正確的施作流程，是做出持久漂亮凝膠甲的關鍵：

**步驟 1：前置處理**
清潔雙手 → 推皮 → 輕磨甲面去除光澤 → 上甲面處理液 → 等待揮發

**步驟 2：底膠（Base Coat）**
薄薄一層，覆蓋整個甲面 → 照燈固化（LED 30–60 秒）

**步驟 3：色膠（第一層）**
均勻薄塗，**不要堆積** → 照燈固化

**步驟 4：色膠（第二層，視需要第三層）**
加強顏色飽和度 → 照燈固化

**步驟 5：封層（Top Coat）**
覆蓋整個甲面並**包邊**（包覆甲尖前端）→ 照燈固化 → 用清潔液擦除殘留層（如需要）

## 四個新手必知技巧

1. **薄塗多層，不要一次堆厚**：每層塗太厚容易起皺、固化不均，薄層分次塗才是正確做法。

2. **包邊不可省**：封層時一定要用筆刷沿著甲尖「包邊」一圈，是防止甲尖最先磨損的關鍵步驟。

3. **保持距離甲緣**：凝膠碰到皮膚後，固化時會從皮膚開始翹起，導致翹甲。塗膠時距甲緣保持 0.5–1mm 的安全距離。

4. **確保每層完全固化**：照燈時間不夠會讓凝膠殘留黏手感，後續步驟也會混色。每層都要足量照燈，不要怕浪費時間。

## 做完後的保養提醒

凝膠甲完成後，日常保養決定了它能維持多久。每天使用指緣油滋潤、洗碗時戴手套、避免用指甲當工具，這些小習慣能讓凝膠甲的壽命從兩週延長到三週以上。
