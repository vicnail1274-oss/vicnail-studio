## 美甲燈的原理

美甲燈發出特定波長的 UV 或 LED 光線，啟動凝膠中的光起始劑（photoinitiator），使凝膠硬化。若照燈不足，凝膠會保持黏軟，容易暈開或翹邊。

**來源：** 美容皮膚病學期刊，2025 — https://onlinelibrary.wiley.com/journal/14732130

---

## UV 燈 vs LED 燈：核心差異

| 項目 | UV 燈 | LED 燈 |
|------|--------|--------|
| 波長 | 320–400 nm（全波長 UV-A）| 385–405 nm（窄波 UV-A）|
| 照燈時間 | 60–120 秒 | 10–30 秒 |
| 燈泡壽命 | 50–100 小時 | 5,000–50,000 小時 |
| 凝膠相容性 | 所有凝膠皆可固化 | 僅限 LED 相容凝膠 |
| 發熱風險 | 較低 | 可能較高（過度照燈有灼傷風險）|
| 價格區間 | $10–$50 | $20–$80 |
| 含汞 | 是 | 否 |

---

## LED 燈可以固化所有凝膠嗎？

**不行。** 許多凝膠（尤其是平價品牌、特定顏色：白色、紅色、深藍色）使用的光起始劑需要 365nm UV 波長才能反應，常見 LED 燈無法激發。

2026 年大致原則：
- 透明凝膠、定色膠 → LED 可固化
- 白色、霓虹色、深紅凝膠 → 通常需要 UV 或雙光源 UV/LED 燈
- 建構膠 / 硬膠 → UV 或高功率 LED（48W 以上）較穩定

**來源：** CND（Creative Nail Design）2026 技術公告 — https://www.cnd.com/technicians/lamp-guide

## 瓦數比燈泡類型更重要

2026 年市場趨勢是高瓦數：

| 瓦數 | 適用場景 | 固化時間 |
|------|---------|---------|
| 6–9W | 旅行 / 緊急備用 | 60–120 秒 |
| 12–18W | 居家使用、薄膠層 | 30–60 秒 |
| 24–36W | 一般居家 / 美甲店 | 15–30 秒 |
| 48W 以上 | 專業級、建構膠、快速服務 | 5–15 秒 |

**關鍵數據：** 48W LED 燈比 9W UV 燈固化更快、更均勻。2026 年多數專業沙龍已升級至 48W–72W LED 燈。

## UV 安全性：你需要擔心嗎？

美甲燈的 UV 暴露量遠低於日光機，但多年累積使用已引發關注：

- 單次凝膠美甲 ≈ 5–20 分鐘 UV-A 暴露
- 皮膚科醫生建議：照燈前雙手塗抹 SPF 30+ 防曬，或使用露指 UV 防護手套
- FDA 2025 年聲明：目前無明確證據顯示美甲燈與皮膚癌關聯，但仍建議預防措施

**來源：** FDA 消費者更新，2025 — https://www.fda.gov/consumers/consumer-updates/tattoos-piercings/nail-lamp-safety

## 正確使用美甲燈的方式

1. **薄層塗抹凝膠** — 厚層無法均勻固化
2. **每層完全固化後**再塗下一層
3. **包覆游離緣** — 防止指尖崩裂
4. **不要提前取出** — 強制提前會導致固化不足
5. **最後一次固化後**，用 91% 以上異丙醇擦除殘餘黏膠

---

## 燈具固化不足的跡象

- 照燈後仍黏手
- 顏色可用棉片擦除
- 凝膠 2–3 天內翹邊
- 指甲感覺黏軟

解決方式：檢查瓦數、更換 UV 燈泡（UV 款）、延長照燈時間，或升級高功率燈具。

---

## 總結

1. LED 燈固化更快（10–30 秒）且壽命長；UV 燈可固化所有凝膠
2. 購買前確認凝膠是否為「LED 相容」或「UV/LED」
3. 48W 以上是 2026 年居家與專業的主流標準
4. 美甲燈 UV 暴露量低，但建議塗防曬作為預防
5. UV 燈泡每 50–100 小時需更換；LED 燈可使用數年
