很多人做完凝膠延甲後都有同樣的困擾：形狀不夠自然、甲面看起來太厚、甲邊翹起或是一兩週就斷裂。這些問題的根源，幾乎都出在**塑型技巧**上，而不是膠的品質。

凝膠延甲的核心不是「堆多少膠」，而是「怎麼堆、堆在哪裡」。掌握 5 個關鍵技巧，你的延甲就能同時擁有好看的外形和足夠的強度。

---

## 為什麼塑型技巧這麼重要？

凝膠延甲是以建構膠（builder gel）或 BIAB 膠為基礎，在甲片或甲模上延伸甲面長度。延長後的指甲**受力方式和自然甲完全不同**，如果膠的分佈不正確，就會造成：

- 甲面受力不均 → 容易從中間斷裂
- 甲根部膠層過厚 → 長出新甲後分離感明顯
- 延伸段太薄 → 彎折後直接碎裂

正確的塑型技巧能讓膠的厚度符合指甲結構的力學需求，延甲才會既好看又耐用。

---

## 5 個關鍵塑型技巧

### 技巧 1：先確定延伸長度和形狀

在上膠之前，先用甲片（tip）或甲模（nail form）確認好你想要的形狀。常見的凝膠延甲形狀：

| 形狀 | 特色 | 適合誰 |
|------|------|--------|
| 方形（Square） | 稜角俐落、視覺加寬 | 手指細長的人 |
| 橢圓形（Oval） | 自然柔和、延伸感強 | 各種手型通用 |
| 尖形（Stiletto） | 戲劇感強，難度高 | 進階技術 |
| 杏仁形（Almond） | 優雅中庸、最熱門 | 想兼顧美感和耐用 |

> **提醒**：確定形狀後再開始施膠，中途改形狀會讓膠層結構混亂。

### 技巧 2：建立正確的 C 型曲線（最重要）

C 型曲線（C-curve）是指從指甲橫截面看到的拱形弧度。這個弧度是延甲強度的關鍵。

**標準 C 型曲線：截面弧度約 20–35%**

- 曲線不夠 → 延甲容易「側壓斷」
- 曲線過大 → 造成甲邊夾緊、不舒服

建立 C 型曲線的時機是在**膠尚未固化之前**，用食指和拇指輕輕夾住甲面兩側塑形，再送入燈管照射固化。

### 技巧 3：分層施膠，控制各區厚度

延甲的膠厚不是均一的。正確的分層邏輯如下：

```
甲根（自然甲起始點）：最薄（0.5–0.8mm）
甲面中段（應力點）：最厚（1.0–1.5mm）
延伸段（甲尖）：薄而逐漸收尖（0.3–0.5mm）
```

**為什麼中段要最厚？**
中段是延甲受力最集中的位置（stress area），加厚中段可以分散壓力，防止「從中間斷」的常見失敗。

### 技巧 4：甲面整形要一次到位

凝膠固化後用磨棒整形，這個步驟要一次做好，避免反覆磨——每磨一次就會破壞甲面結構，讓延甲越來越薄。

整形重點：
1. **先磨側面**，確認兩側對稱
2. **再磨頂面**，統一厚度
3. **最後磨甲尖**，修出你想要的形狀
4. 用 180 號磨棒收尾，表面粗糙度均勻才能讓封層貼合

### 技巧 5：封層不可省，厚度要夠

很多人延甲後只上一層薄薄的封層，這樣無法保護凝膠層。建議：

- **第一層封層**：塗薄、確保覆蓋全面
- **第二層封層**：加厚甲尖邊緣（包邊），防止翹起
- 可選：UV 強化封層（high-gloss top coat）增加亮度和硬度

## 常見失誤 vs 正確做法

| 常見失誤 | 問題 | 正確做法 |
|---------|------|---------|
| 甲根膠堆太厚 | 長出新甲後分離快 | 甲根保持薄而平滑的過渡 |
| 延伸段膠太少 | 甲尖容易碎 | 甲尖適當包膠，確保強度 |
| 沒做 C 型曲線 | 容易側壓斷 | 照燈前先夾住塑形 |
| 封層只包甲面 | 甲邊翹起 | 封層包邊，覆蓋甲尖側面 |

---

## 延甲後的保型維護

做好延甲不代表一勞永逸。以下是讓塑型維持更久的日常習慣：

- **避免用甲尖開拉環**：用手指腹代替甲尖出力
- **戴手套做家事**：長時間接觸水和清潔劑會讓延甲軟化
- **每 2–3 週回去補甲**：讓師傅重新整理甲根，維持結構完整

## 小結

凝膠延甲的塑型技巧可以用一句話總結：**在對的位置放對的厚度，做出正確的 C 型曲線**。

掌握這 5 個技巧，不論你是剛開始學延甲的新手，還是想提升作品品質的美甲師，都能明顯感受到延甲的耐用度和美觀度提升。

下次預約延甲前，不妨和你的美甲師討論一下你喜歡的形狀和長度——了解塑型邏輯的客人，更能和師傅合作出最理想的成品。
