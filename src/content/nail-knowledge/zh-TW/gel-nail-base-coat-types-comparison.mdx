底膠是凝膠美甲的地基，選錯底膠不只影響持久度，還可能造成起翹、斷裂或傷甲。市面上底膠種類繁多，「軟膠」「硬膠」「橡皮底膠」這些詞令人眼花撩亂。本篇一次釐清所有主要底膠類型，讓你清楚知道該用哪一種。

---

## 底膠的共同功能

無論哪一種底膠，基本功能都包括：

- **增加凝膠附著力**，讓色膠不易起翹
- **保護天然甲**，隔離色素染色
- **提供結構支撐**（部分類型）

差異在於**柔韌度、厚度、適用場景**。

---

## 類型一：軟膠底膠（Soft Gel Base Coat）

### 特性
- 固化後保留一定彈性，不會完全硬化
- 薄透，質地輕盈
- 跟隨天然甲的自然彎曲變形

### 適合對象
- **健康、自然甲**的日常凝膠美甲
- 指甲本身強度正常，不需要額外加固
- 喜歡「貼合感」、不喜歡沉重感的使用者

### 上膠方式
1. 清潔甲面，輕磨打底
2. 薄塗一層軟膠底膠，距甲皮留 0.5mm
3. LED 燈照射 30-60 秒
4. 直接上色膠，不需清潔浮膠

### 注意事項
- 薄甲、易斷甲使用軟膠底膠保護力有限，建議升級橡皮底膠
- 若甲面出油嚴重，建議搭配去油劑（Dehydrator）預處理

---

## 類型二：硬膠底膠（Hard Gel Base Coat）

### 特性
- 固化後接近堅硬狀態，強度高
- 較厚，具有明顯的結構支撐感
- 不溶於去光水，只能以銼磨方式卸除

### 適合對象
- **脆弱、薄弱的天然甲**，需要強力支撐
- 習慣用指甲做重活的人
- 想要做指甲延伸（加長）的情境

### 上膠方式
1. 甲面打磨清潔
2. 塗一層硬膠底膠，可稍厚
3. LED 照燈 60 秒
4. 用高目數磨甲棒輕磨表面，再上色膠

### 注意事項
- 硬度高但彈性差，若指甲受到衝擊，較容易整片斷裂而非彎曲緩衝
- 卸除需時間較長，必須用銼磨而非浸泡

---

## 類型三：橡皮底膠（Rubber Base Coat）

橡皮底膠是近年最受美甲師推崇的底膠類型，介於軟膠和硬膠之間。

### 特性
- **高彈性、高附著力**，橡皮質感讓色膠更牢固貼合
- 自帶一定厚度，可輕微填平甲面凹凸
- 可以直接上色膠，也可以作為加固層單獨使用
- 溶於去光水，可正常浸泡卸除

### 適合對象
- **脆甲、薄甲、易起翹**的問題指甲
- 手部活動量大（長時間打字、手工藝）
- 希望凝膠更持久、減少邊緣起翹的使用者
- 初學美甲師的首選，因為容錯率高

### 上膠方式
1. 清潔甲面，去油處理
2. 塗一層橡皮底膠（可比軟膠稍厚）
3. LED 照燈 60 秒
4. 用清潔棉片稍微清潔浮膠（部分品牌需要）
5. 直接上色膠

### 特殊用法：厚塗橡皮底膠
部分美甲師會將橡皮底膠厚塗 2-3 層，雕塑出輕微弧度，替代建構膠使用，適合想要微加固但不想做完整建構的情況。

---

## 類型四：建構底膠（Builder Gel Base / BIAB）

BIAB（Builder in a Bottle）是近年英國和歐洲流行的類型，結合底膠與輕建構功能。

### 特性
- 可以自然流平，填補指甲表面不平整
- 一瓶搞定：底膠 + 輕建構 + 加固
- 比橡皮底膠更厚，需要更長燈照時間

### 適合對象
- 指甲有明顯凹凸、脊線或分層問題
- 想省略多層步驟、追求精簡流程
- 軟甲但不想做延伸，只需輕度加固

## 如何選擇適合自己的底膠

1. **健康自然甲，只是想做漂亮凝膠** → 軟膠底膠
2. **指甲薄、易斷、容易起翹** → 橡皮底膠（首選）
3. **需要指甲延伸或極度脆弱** → 硬膠底膠
4. **指甲有凹凸不平，想要精簡步驟** → 建構底膠
